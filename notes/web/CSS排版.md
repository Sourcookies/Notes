标准流版型 浮动版型 定位版型



块级元素：不能并排
行内元素（内联）元素：横向排列



浮动排版：float
float : left
浮动元素脱离文档流，浮动后的子元素无法撑开父元素
元素由两层组成：元素层 内容层



浮动清除：

```
clear属性的元素将下沉到浮动元素的后面
```



定位排版：

```
position:
static
relative	相对定位
absolute	绝对定位
fixed
```

解决**层级**之间的叠加现象



相对定位 ：

相对于**自身原始位置偏移**

原占的空间仍然保留，不会脱离文档流

```
display:
none			隐藏元素
block			显示
inline-block	非行内块元素-> 行内块元素
inline			非行内元素-> 行内元素
```

绝对定位：

相对于自己最近的有定位的祖先元素，脱离文档流
